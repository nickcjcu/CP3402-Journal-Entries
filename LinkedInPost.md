# LinkedIn Post Content

Over the past few weeks, I’ve been learning the fundamentals of web development and content management systems through building my own project, ActionBoost.

I started with basic HTML, CSS, and Bootstrap to create a simple static website. This helped me understand how pages are structured and how navigation works.

I then moved on to working with CMS platforms. I built sites using both Joomla and WordPress, which helped me understand different approaches to managing content. Joomla felt more structured with articles and modules, while WordPress was more user-friendly with pages and themes.

I also learned how to set up a local development environment using LocalWP. This allowed me to run a dynamic website with a database on my own computer, which gave me a better understanding of how backend systems support websites.

More recently, I worked on creating a WordPress child theme. This taught me how to customise a website safely without modifying the original theme, and how small design changes can impact the overall user experience.

One key thing I’ve learned is that building and testing things myself is the most effective way to learn. Writing weekly learning journals also helped me reflect on what I’ve understood and improve how I explain technical concepts.

Overall, this experience has given me a strong foundation in how websites are built, managed, and deployed, and has improved both my technical and problem-solving skills.

## Content Insights
PHP allows websites to generate dynamic content by combining programming logic with HTML. Using `echo` makes it possible to output headings, paragraphs, and lists directly into the webpage. Conditional statements such as `if/else` allow the website to make decisions, for example displaying “Pass” or “Fail” depending on a score.

Loops help automate repetitive tasks. A `for` loop can iterate a set number of times, a `while` loop runs while a condition is true, and `foreach` is ideal for processing items in an array. Functions make code reusable and easier to manage, especially when parameters allow the same function to produce different results.

Using `include` for the header and footer showed me how modular code improves maintainability. Instead of repeating the same code on multiple pages, the common elements are kept in separate files and included where needed. This is a key practice in professional web development.

## Career/Employability/Learning Insights
Learning PHP is a practical skill for backend web development, as it is widely used for creating dynamic websites and interacting with databases. Understanding loops, conditional logic, functions, and modular code prepares me for real-world development tasks.

This prac also reinforced problem-solving and testing skills. Writing original code, debugging errors, and experimenting with loops and functions helped me build confidence in programming logic. It highlighted the importance of breaking tasks into small steps and testing each piece, a method that translates directly to professional coding practices.

Overall, this week strengthened my ability to create dynamic, interactive web pages and gave me a solid foundation for further backend development tasks in future projects.

# Week 2

## Learning Activities & Resources
This week, I focused on learning how to use Joomla CMS for the first time. I set up my Joomla site using CloudAccess.net, which provided an easy way to install and host the CMS online. I initially tried Joomla version 6 with the Gantry “Gnarly” template, but it was incompatible, so I switched to Joomla 5.4 where the template worked correctly. My main learning task was to understand how Joomla articles, modules, and menus work together to create a functioning website. I spent time creating articles for the homepage, about us page, and features page, and learning how to link them properly in the navigation menu so that visitors could navigate between pages.  

To support my learning, I watched multiple YouTube tutorials that explained Joomla site structure, templates, and menus, reviewed the course materials that provided guidance on CMS concepts, and read articles on template compatibility and basic Joomla best practices. I experimented with creating and publishing content, editing menus, and arranging modules to see how they affected the website layout. Initially, Joomla was confusing because of the number of components and settings, but following tutorials while practising on the live site helped me gradually understand how everything fits together. This hands-on experimentation allowed me to see how changes in articles or menus affected the visible website immediately, which was a very useful way to learn.

## Content Insights
Using the Gantry theme “Gnarly” was a good introduction to learning how to use Joomla elements without having to build a theme from scratch. Working with CloudAccess.net helped me understand the practical aspects of deploying and managing a Joomla site online. I learned the structure of Joomla articles, how they can be created, edited, and linked to menus, and how modules allow content or features to appear in different parts of the site. I realised that linking articles correctly in the navigation bar is crucial for user experience and that small mistakes in linking can make content invisible or hard to find. Experimenting with modules, templates, and menu settings helped me understand how Joomla separates content management from presentation, which is an important concept for building maintainable websites.  

I also learned that persistence and careful experimentation are key when working with a CMS for the first time. Some features were unintuitive at first, like menu linking or module placement, but by repeatedly trying, observing the results, and consulting tutorials, I gradually understood how to control the site. This reinforced that learning complex systems often requires a mix of guided instruction and hands-on practice.

## Career/Employability/Learning Insights
This week highlighted important skills for employability, such as problem-solving, adaptability, and self-directed learning. Encountering template compatibility issues and CMS complexity taught me to research solutions, try multiple approaches, and adapt when the first attempt does not work. Practising how to structure content and link pages in a CMS is directly relevant to web development roles, content management positions, and IT support jobs. I also gained confidence in navigating hosting platforms like CloudAccess.net, which shows I can manage online environments, troubleshoot issues, and make informed decisions about CMS setup. Combining tutorial-based learning with real-world experimentation reinforced my ability to learn independently, a skill that will help me in future projects and in a professional workplace.
